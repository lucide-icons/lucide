# Lucide React Native

Implementation of the lucide icon library for React Native applications.

## Installation

```sh
pnpm add lucide-react-native
```

```sh
npm install lucide-react-native
```

```sh
yarn add lucide-react-native
```

```sh
bun add lucide-react-native
```
