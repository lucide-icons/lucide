# Lucide

Implementation of the lucide icon library for web applications.

## Installation

### Package Managers

```sh
npm install lucide
```

```sh
pnpm install lucide
```

```sh
yarn add lucide
```

### CDN

```html
<!-- Development version -->
<script src="https://unpkg.com/lucide@latest/dist/umd/lucide.js"></script>

<!-- Production version -->
<script src="https://unpkg.com/lucide@latest"></script>
```
