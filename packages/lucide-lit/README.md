# Lucide Lit

Implementation of the Lucide icon library for [Lit](https://lit.dev/) applications. Each icon is a custom element (for example `<lucide-camera>`) built with `LitElement`.

## Installation

```sh
pnpm add lucide-lit lit
```

```sh
npm install lucide-lit lit
```

```sh
yarn add lucide-lit lit
```

```sh
bun add lucide-lit lit
```
