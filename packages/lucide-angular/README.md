# Lucide Angular

Implementation of the lucide icon library for angular applications.

## Installation

```sh
pnpm add lucide-angular
```

```sh
npm install lucide-angular
```

```sh
yarn add lucide-angular
```

```sh
bun add lucide-angular
```
