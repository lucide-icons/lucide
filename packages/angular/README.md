# Lucide Angular

A standalone, signal based, zoneless implementation of the Lucide icon library for Angular applications.

## Installation

```sh
pnpm add @lucide/angular
```

```sh
npm install @lucide/angular
```

```sh
yarn add @lucide/angular
```

```sh
bun add @lucide/angular
```
