# Lucide Solid

Implementation of the lucide icon library for solid applications.

> What is lucide? Read it [here](https://github.com/lucide-icons/lucide#what-is-lucide).

## Installation

```sh
yarn add lucide-solid
```

or

```sh
npm install lucide-solid
```
