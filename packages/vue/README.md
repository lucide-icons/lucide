# Lucide Vue

Implementation of the lucide icon library for Vue 3 applications.

## Installation

```sh
pnpm add @lucide/vue
```

```sh
npm install @lucide/vue
```

```sh
yarn add @lucide/vue
```

```sh
bun add @lucide/vue
```
